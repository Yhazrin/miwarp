# MiWarp Setup Wizard

Welcome to MiWarp! This wizard will guide you through setting up your workspace in a few simple steps.

## Step 1 — Role

Let's start by understanding what kind of work you do. This helps us customize MiWarp for your needs.

**Quick intro:** MiWarp wraps AI coding tools (Claude Code, Codex) with a visual chat interface, session management, and activity monitoring. Everything is local-first — your data stays on your machine.

Select your role from the options below:
- 💻 Developer
- 🎨 Designer
- 📦 Product Manager
- 🔬 Researcher
- ✍️ Writer
- ⚡ General Productivity

Once you select your role, we'll suggest a plugin that matches your workflow.

---

## Step 2 — Plugin

Great! Now let's install a plugin that fits your work.

Plugins bundle skills for your role — reusable workflows you trigger with `/name`. For example:
- `/schedule` — Create automated tasks
- `/review` — Run code reviews
- `/init` — Initialize new projects

We'll suggest the best plugin for your role. You can always explore more in the Skill Marketplace later.

---

## Step 3 — Try a Skill

Skills are the heart of MiWarp. They are reusable workflows that automate repetitive tasks.

Try one now:
- **/schedule** — Create your first scheduled task
- **/consolidate-memory** — Organize your memory files
- **/review** — Review your current project
- **/init** — Initialize a new project structure

Pick one and run it. Skills execute in the background and return results to you.

---

## Step 4 — Connectors

Connectors plug in your actual tools so skills have real context:
- 📧 Email (Gmail, Outlook)
- 📅 Calendar (Google Calendar, Outlook)
- 📝 Notes (Notion, Obsidian)
- 💬 Communication (Slack, Teams)
- 📋 Tasks (Jira, Linear, Asana)
- 🐙 Code (GitHub, GitLab)

Connect the tools you use daily to unlock powerful automation.

---

## Step 5 — Done

You're all set! Here's a quick summary of what's available:

### Skills
Type `/` in any chat to see your available skills. Try `/schedule`, `/review`, or `/init`.

### Plugins
Browse the Skill Marketplace for domain-specific plugins. Visit `/plugins` to explore.

### Scheduled Tasks
Automate repetitive work by scheduling tasks. Try `/schedule` to create your first one.

### Multi-Agent
Run multiple AI sessions in parallel for complex tasks. Visit `/multi-agent` to learn more.

---

## Quick Tips

- **Keyboard shortcuts**: Press `Cmd+K` to open command palette
- **Team collaboration**: Invite teammates in `/teams`
- **Memory**: Your context persists across sessions
- **Settings**: Customize your experience in `/settings`

---

## Ground Rules

1. **One step at a time** — Don't rush, each step builds on the previous
2. **Skips are fine** — If a step doesn't apply to you, move on
3. **Return to flow** — After trying something, come back to continue setup
4. **Ask for help** — Type `/help` anytime for assistance
