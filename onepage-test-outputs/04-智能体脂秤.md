### 核心卖点

- **【核心卖点一】双频BIA：医疗级精准测量**
  - **领先性**：双频BIA技术（50kHz+500kHz），测量精度±0.5%，业内领先
  - **利益点**：150项身体数据（体脂率/肌肉量/内脏脂肪/骨量/水分等），数据可靠可指导训练
  - **竞对能力**：华为体脂秤3 Pro单频BIA，精度±1%；Withings双频但价格¥899
  - 效果对比：

    | 对比维度 | 单频BIA（普通体脂秤） | 双频BIA（S400） |
    |---|---|---|
    | 测量精度 | ±1-2% | ±0.5% |
    | 数据维度 | 8-12项 | 150项 |
    | 内脏脂肪检测 | 不支持 | 支持 |
    | 肌肉量分段检测 | 不支持 | 支持（四肢+躯干） |

- **【核心卖点二】数据追踪：米家APP趋势分析+AI建议**
  - **领先性**：接入米家生态，数据自动同步，AI生成训练/饮食建议
  - **利益点**：每日称重自动生成趋势图，体重/体脂/肌肉量变化一目了然
  - **竞对能力**：Withings有APP但生态封闭；华为健康数据互通性一般
  - 效果对比：

    | 对比维度 | 传统体脂秤 | 小米体脂秤S400 |
    |---|---|---|
    | 数据记录 | 手动记录 | 自动同步米家APP |
    | 趋势分析 | 无 | 日/周/月趋势图 |
    | AI建议 | 无 | 训练/饮食个性化建议 |
    | 多人管理 | 不支持 | 支持16人分组 |
