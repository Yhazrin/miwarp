> **米家智能台灯Pro——护眼+场景联动，学生和办公人群的智能桌面光源。**

  - 国内市场约50亿元，明基/松下/米家三足鼎立

### 核心卖点

- **【核心卖点一】专业护眼：国AA级照度+Ra98高显色**
  - **领先性**：国AA级照度标准，Ra98高显色指数，通过德国莱茵TÜV护眼认证
  - **利益点**：长时间用眼不疲劳，色彩还原真实，适合阅读/绘画/设计
  - **竞对能力**：明基WiT同级别护眼但价格¥1299；松下护眼台灯Ra95
  - 效果对比：

    | 对比维度 | 普通台灯 | 米家智能台灯Pro |
    |---|---|---|
    | 照度等级 | A级 | 国AA级 |
    | 显色指数 | Ra80-85 | Ra98 |
    | 频闪 | 有频闪 | 无频闪 |
    | 蓝光危害 | RG1 | RG0（豁免级） |

- **【核心卖点二】场景联动：米家生态智能控制**
  - **领先性**：接入米家生态，支持小爱语音/APP/自动感应多模式控制
  - **利益点**：到家自动亮灯、离座自动关灯、学习模式自动调光、睡眠模式渐暗
  - **竞对能力**：明基无智能家居联动；松下仅支持基础APP控制
  - 效果对比：

    | 对比维度 | 传统智能台灯 | 米家智能台灯Pro |
    |---|---|---|
    | 控制方式 | 按键/APP | 语音+APP+自动感应+场景联动 |
    | 离座感应 | 不支持 | 支持（雷达感应） |
    | 场景模式 | 3-5种 | 自定义无限种 |
    | 生态联动 | 无 | 米家全屋联动 |

- **【核心卖点三】自适应调光：环境光感应+AI算法**
  - **领先性**：内置环境光传感器+AI调光算法，自动匹配最佳亮度和色温
  - **利益点**：无需手动调节，不同时间段自动匹配最舒适光线
  - **竞对能力**：明基有环境光感应但无AI算法；松下无此功能
