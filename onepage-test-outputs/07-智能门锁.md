### 核心卖点

- **【核心卖点一】3D人脸识别：解放双手，回家只需看一眼**
  - **领先性**：3D结构光人脸识别，活体检测防照片/视频攻击，0.3秒极速解锁
  - **利益点**：双手提东西也能开门，老人小孩指纹不清晰也能用，夜间红外也能识别
  - **竞对能力**：德施曼有3D人脸但价格¥2999+；凯迪仕中端款仅2D人脸
  - 效果对比：

    | 对比维度 | 2D人脸识别 | 3D结构光（X20） |
    |---|---|---|
    | 解锁速度 | 0.5-1秒 | 0.3秒 |
    | 安全性 | 照片可破解 | 活体检测，照片/视频无效 |
    | 夜间识别 | 差 | 红外辅助，全黑暗可用 |
    | 佩戴口罩 | 不支持 | 支持（眼部特征识别） |

- **【核心卖点二】猫眼大屏：4.7英寸高清大屏+远程可视对讲**
  - **领先性**：4.7英寸IPS高清大屏，160°广角猫眼，远程可视对讲
  - **利益点**：门口有人按铃，手机/大屏都能看；快递放门口实时监控；老人小孩也能看清访客
  - **竞对能力**：德施曼有猫眼但屏幕小（3.5寸）；凯迪仕无远程对讲
  - 效果对比：

    | 对比维度 | 传统猫眼 | X20智能猫眼 |
    |---|---|---|
    | 屏幕尺寸 | 无/小屏 | 4.7英寸高清 |
    | 视角范围 | 90-120° | 160°广角 |
    | 远程查看 | 不支持 | APP实时查看 |
    | 可视对讲 | 不支持 | 支持（双向语音+视频） |
    | 夜视 | 红外补光 | 红外+微光全彩 |

- **【核心卖点三】全自动锁体：关门即锁，虚掩报警**
  - **领先性**：全自动锁体，关门自动上锁，虚掩/未关实时报警推送
  - **利益点**：再也不用回头确认是否锁门，虚掩立即手机提醒
  - **竞对能力**：半自动锁体需手动上锁，行业内全自动锁体占比仅40%
