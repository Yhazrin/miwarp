### 核心卖点

- **【核心卖点一】压力感应：防止过度刷牙损伤牙龈**
  - **领先性**：内置压力传感器，实时监测刷牙力度，超压自动降速+LED警示
  - **利益点**：保护牙龈不受伤，养成正确刷牙习惯，牙龈出血率降低40%
  - **竞对能力**：飞利浦HX6730有压力感应但无APP联动；欧乐B中端款无此功能
  - 效果对比：

    | 对比维度 | 无压力感应（普通电动牙刷） | 有压力感应（T700） |
    |---|---|---|
    | 刷牙力度控制 | 凭感觉 | 实时监测+自动调节 |
    | 牙龈保护 | 易过度用力 | 超压自动降速 |
    | 刷牙习惯 | 无反馈 | APP数据反馈+改进建议 |

- **【核心卖点二】APP智能指导：刷牙报告+个性化方案**
  - **领先性**：6轴陀螺仪追踪刷牙轨迹，APP生成16区刷牙覆盖报告
  - **利益点**：每次刷牙都有"成绩单"，漏刷区域一目了然，刷牙效果可量化
  - **竞对能力**：飞利浦有APP但数据维度少；欧乐B中端无APP功能
  - 效果对比：

    | 对比维度 | 传统电动牙刷 | 米家T700 |
    |---|---|---|
    | 刷牙反馈 | 无 | 16区覆盖报告 |
    | 时间管理 | 2分钟定时 | 分区计时+漏刷提醒 |
    | 历史记录 | 无 | 30天趋势分析 |
    | 个性化建议 | 无 | AI生成刷牙方案 |

- **【核心卖点三】清洁力：40000次/分钟高频振动+磁悬浮马达**
  - **领先性**：磁悬浮声波马达，40000次/分钟高频振动，清洁效率提升30%
  - **利益点**：齿缝清洁更彻底，牙菌斑去除率99%，亮白效果可见
  - **竞对能力**：飞利浦同频段但价格¥399+；欧乐B旋转式噪音大
