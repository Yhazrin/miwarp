## 二、核心发现速览

### P0 必备（没有就不买）

| # | 发现 | 证据强度 | 定量验证 |
|---|------|---------|---------|
| 1 | **预览+自动对焦是购买底线** — 所有参与者都将"预览功能"和"自动对焦"列为"没有就不考虑"的必备功能 | HIGH | KANO中端必备含"翻转屏自拍"；屏幕满意度中端0.483 vs 高端0.739，差异显著 |
| 2 | **相纸成本是行为改变的关键杠杆** — 1.98元/张 vs 富士8-9元/张，直接改变了用户"不敢按快门"的心理 | HIGH | 相纸满意度中端0.468 vs 高端0.605，中端用户更敏感 |
| 3 | **一次出片是体验底线** — 用户明确拒绝"来回来去三次"的出片方式 | HIGH | 全场一致 |

### P1 重要（差异化卖点）

| # | 发现 | 证据强度 | 定量验证 |
|---|------|---------|---------|
| 4 | **"色彩效果"≠高饱和度，而是"有质感的色彩把控"** — 用户对色彩效果的理解与产品团队存在偏差 | MEDIUM | KANO"滤镜色彩效果"为中端必备；成像满意度中端0.505 vs 高端0.579 |
| 5 | **AR动态照片是"超出预期的价值点"** — 多人自发提到AR功能"特别神奇，有意思" | HIGH | KANO中端+高端均列为必备功能 |
| 6 | **品牌是Top1考虑因素（32.8%）** — 但中端(33.5%)与高端(32.3%)差异不大 | HIGH | 定量问卷900份 |

### P2 增值（有更好）

| # | 发现 | 证据强度 | 定量验证 |
|---|------|---------|---------|
| 7 | 群像拍摄能力（4-6人）是场景缺口 | HIGH | 多人合影频率70.1%（Top1），满意度仅0.626——高频低满意 |
| 8 | 快门交互反馈需要"干脆感" | LOW | 无定量数据 |

| **表层** | 用户觉得AR照片好玩 |
| **中层** | 因为AR照片将"静态记忆"升级为"动态记忆"，满足了用户对"承载更多记忆和纪念"的情感需求 |
| **深层** | "若干年之后，你再把它扫出来，哇，那一刻心情真的是鸡皮疙瘩都起来了"——AR照片创造了"时间胶囊"效应，与拍立得的"记忆载体"本质高度契合 |
| **证据** | E025相关用户原话；KANO中端+高端均必备 |
| **决策影响** | AR动态照片应作为核心卖点重点宣传，而非"附加功能" |
| **建议行动** | 1. 将AR照片作为产品发布会的核心演示环节；2. 考虑"AR照片模板持续上新"作为长期用户粘性策略；3. 工程样机体验测试+情感反应量表 |
| **风险** | 如果AR体验卡顿或效果差，会适得其反 |

### 洞察卡4：便携性是"保健因素"而非"激励因素"

| 维度 | 内容 |
|------|------|
| **洞察** | 便携性在定性中被高频自发提及，但定量Top1仅4.9%——这是"有了不觉得好，没有就觉得差"的保健因素，而非"有了更满意"的激励因素 |
| **表层** | 用户要小巧轻便的拍立得 |
| **中层** | 因为"宠物开的非常多，很难抓拍"，需要"随时随地背一个" |
| **深层** | 便携性的本质是"无感携带"——不是越薄越好，而是"放进口袋/小包不觉得负担"。同时需要"有质感"——太薄会显得廉价。定量数据中排名低是因为用户默认"拍立得应该便携"，不构成购买决策因素 |
| **证据** | E014, E016, E017；便携性Top1 4.9%，Top3 30.4% |
| **决策影响** | 产品尺寸应在"口袋级"和"有质感"之间寻找平衡点，但不应作为核心卖点宣传 |
| **建议行动** | 1. 原型测试增加"握感"维度；2. 材质选择上，牛奶白+细腻纹理 > 极致薄+光滑塑料；3. 不要将便携性作为主要营销点 |
| **风险** | 如果为了极致薄牺牲了握感和质感，用户会降级评价 |

### 洞察卡5：褪色周期是成像选择的隐性约束

| 维度 | 内容 |
|------|------|
| **洞察** | 用户在选择成像技术时，不仅看即时效果，还会权衡褪色周期——"三年前已经在退了，变灰了" |
| **表层** | 用户关心照片能保存多久 |
| **中层** | 因为拍立得的核心价值是"记忆载体"，褪色等于"记忆丢失" |
| **深层** | 用户对拍立得的期望是"永久保存"，但现实中所有银盐相纸都会褪色。LED光敏成像如果能提供更长保质期，是差异化优势 |
| **证据** | E021, E022 |
| **决策影响** | 如果LED光敏成像的保质期优于银盐技术，应作为差异化卖点重点宣传 |
| **建议行动** | 1. 工程团队确认LED光敏成像的褪色周期数据；2. 如有优势，在产品说明中标注"保质期XX年"；3. 褪色加速实验（高温高湿条件） |
| **风险** | 如果保质期数据不如预期，不要主动提及 |

### 洞察卡6：群像拍摄是"高频低满意度"的优先改善点

| 维度 | 内容 |
|------|------|
| **洞察** | 多人合影是使用频率最高的拍摄内容（70.1%），但满意度仅0.626——是典型的"高频低满意度"痛点，改善ROI最高 |
| **表层** | 用户希望拍群像效果好 |
| **中层** | 因为朋友聚会、生日等场景高频需要群像，但当前拍立得群像效果差 |
| **深层** | 定量数据佐证：多人合影使用频率70.1%（Top1），但满意度仅0.626——是"高频低满意度"的典型痛点。改善群像能力的边际收益远高于改善低频场景 |
| **证据** | E033, E034；多人合影频率70.1%，NSS 0.626 |
| **决策影响** | 群像拍摄能力应作为产品差异化方向之一，优先级高于低频场景改善 |
| **建议行动** | 1. 评估广角/拼接/AI补图等技术方案；2. 如果技术可行，在功能测试中重点验证；3. 目标：将群像NSS从0.626提升至0.70+ |
| **风险** | 如果群像效果仍不理想，不要过度承诺 |

### 洞察卡7（新增）：电池续航是满意点，不应列为改善优先级

| 维度 | 内容 |
|------|------|
| **洞察** | 电池及续航NSS 0.675（较高），是用户满意度较好的维度——第一次报告中将续航列为"需要改善（30-40张）"与定量数据矛盾 |
| **表层** | 用户对续航有一定期望 |
| **中层** | 但当前续航已达到用户可接受水平（NSS 0.675） |
| **深层** | 续航不是瓶颈，不应占用产品改善资源。第一次报告中基于座谈会定性观察得出"续航需要改善"的结论，与定量数据矛盾——定性中个别用户的高期望不代表整体用户不满 |
| **证据** | 电池及续航NSS 0.675 |
| **决策影响** | 维持当前续航水平即可，不应列为产品改善优先级 |
| **建议行动** | 1. 维持当前电池规格；2. 将改善资源投入群像拍摄、色彩效果等高优先级方向 |
| **风险** | 无明显风险 |

| 发现 | 定性结论 | 定量数据 | 判断 |

### 研究局限

| 局限 | 影响 | 缓解措施 |

| 3D打印原型 | 材质/重量/颜色均非最终状态 | 外观测试结论需后续验证 |
| 部分定量数据缺失 | 旅游/电池等维度缺少分群NSS | 建议补充问卷分析 |
