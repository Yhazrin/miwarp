# 智能行李箱用户调研项目需求文档

## 一、调研背景及目的

### 【背景】
电动行李箱是一个同时具备科技先锋、实用属性的品类，深深切中"Z世代"对于科技潮流产品的偏好，且其关键技术领域（电机、电池、智能控制）小米均有涉足。当前电商销售正在快速增长，是良好的产品创新标的品类。

在进行产品定义前，相较于传统行李箱，我们期望通过用户调研，建立电动行李箱的已购、潜在购买人群画像认知，了解其购买的关键影响因素、对于传统和电动行李箱的认知差异，辅助市场规模预估、产品定义和切入策略制定，保证在新品类的切入坚守"守正出奇"的基本方略，不出现大方向的偏离。

- 人群要有细分：不同标签人群，特定人群如"Z世代青年"、"千禧一代父母"、"都市白领"等
- 行为要有细分：增购、新购、换购、潜在
- 洞察要有深度：触达→种草→下订→体验→推广

### 【目的】
通过用户调研，建立电动行李箱品类的用户认知体系，为小米切入该品类提供产品定义和市场策略依据。

**优先了解（核心目标）：**
- 电动行李箱已购和潜在购买人群画像，及小米可优先转化的人群
- 用户购买路径、决策链路和关键影响因素
- 用户对电动行李箱的核心需求和功能期望（含Kano模型测试）
- 合理价格区间和功能溢价空间

**其次了解（顺带目标）：**
- 用户对小米品牌做电动行李箱的接受度和期望
- 不同配置/功能组合下的价格敏感度
- 用户使用场景、频率和满意度
- 竞品品牌认知和转化动因

## 二、需解决的问题

### 2.1 人群画像与细分

**核心问题：购买电动行李箱的是什么人？人群特征是？哪些人群是小米可优先转化的？**

1. 人群结构：性别、年龄、城市等级、家庭结构等基础人口统计特征
2. 人群特征：科技观、消费观、购物观，购买行为偏好（如品质、设计、品牌、创新性等）
3. 手机、智能家居品牌持有情况，过往智能硬件产品购买情况，品类关联度（如NAS消费人群关联最多的是数码配件）
4. 不同标签人群的细分画像：Z世代青年、千禧一代父母、都市白领等
5. 已购用户 vs 潜在用户 vs 兴趣用户的人群差异

### 2.2 购买路径与决策链路

**核心问题：用户是怎么了解到的，购买的路径，购买过程遇到的问题，最后是怎么决策的？**

1. 用户购买路径图：需求触发→知道→了解→购买对比→最后决策→购买渠道→实际使用场景→使用体验
2. 第一次看到、被种草、最终被转化的渠道和契机
3. 关键触达渠道细分：短视频、内容平台，测评/直播要细分领域及博主
4. 小米如何触达这些人群、以什么姿势触达和转化？需要提供切入的思路

### 2.3 产品认知与需求定义

**核心问题：用户对行李箱的高端和智能化的认知及要求是什么？购买电动行李箱的动机是？**

1. 用户购买动机是？为什么要选择当前品牌？（设计、功能、品牌等方面举例）
2. 增购、新购、换购的比例分布，前一款产品是什么，什么动机购买电动而非传统产品
3. 购买关注及对比点，哪些是影响用户购买的核心点（如登机限制、使用地点限制、电池/电机安全性等）
4. 功能定位（Kano模型测试）：必备功能、期望功能、魅力功能
5. 一款好的电动行李箱应该具备什么功能

### 2.4 品牌认知与竞争分析

**核心问题：用户对现用行李箱品牌的印象是？如何评价电动与传统行李箱的区别？**

1. 购买之前是否有对比过的品牌，对当前品牌的评价
2. 如何看待电动行李箱与传统行李箱的差异点？
3. 如果小米做这样一款产品，会因为什么点而选择小米？
4. 对小米产品是否有更多的期待，互联、软件功能、硬件配置？

### 2.5 使用体验与痛点

**核心问题：用户的使用过程，使用评价是？满意点及槽点有哪些？**

1. 使用场景分布：机场、车站、小区周边、学校/办公楼、展览会等
2. 使用频率和使用人群（自己/家人/孩子）
3. 满意点和不满意点（骑行性能、智能化、箱体设计、外观、耐用性、细节设计等维度）
4. 用户对未来电动行李箱的期待
5. 哪些场景是现有产品不满足的、做好了能吸引更多用户的
6. 哪些场景是小米可以挖掘新功能体验的

### 2.6 价格测试

**核心问题：合理价格区间是多少？不同配置的溢价空间？**

1. 市面已有产品的定价合理性评估
2. 合适的定价区间，功能溢价测试
3. 假想产品定义：基于本身功能价格，利润空间最大价格，竞争力最大价格
4. 不同价格档位的功能期望差异

## 三、调研形式和人群需求

### 3.1 调研形式

| 形式 | 样本量 | 说明 |
|------|--------|------|
| 定性1v1访谈 | 10人 | 深度了解购买动机、使用体验和需求期望 |
| 外部定量投放 | 1000份（TBD） | 大样本验证人群画像、购买行为和价格敏感度 |

### 3.2 人群需求

涵盖公司内外部用户：

**已购竞品用户：**
- 头部品牌：爱尔威（主京东）
- 其他品牌：DTA、Fevera、UBNI、AOTOS（淘天、拼多多）

**潜在用户：**
- 近期考虑购买，且预算在1000+的用户

**兴趣用户：**
- 对电动行李箱有初步了解但尚未形成购买意向的用户

### 3.3 样本筛选条件

- 年龄：18岁以上（根据法律要求剔除未成年人）
- 已购用户：过去12个月内购买过电动行李箱
- 潜在用户：未来6个月有明确购买意向，预算≥1000元
- 兴趣用户：对电动行李箱有一定了解，有初步兴趣
- 城市分布：一线+新一线城市为主，二线城市为辅
- 家庭月收入：10K以上（或有相应消费能力）

### 3.4 竞品样本覆盖

- 本品品类：电动行李箱/骑行行李箱
- 竞品品类：爱尔威、DTA、Fevera、UBNI、AOTOS、米格堡
- 替代品类：传统行李箱（新秀丽、日默瓦、90分等）
- 购买渠道分布：线上（京东/天猫/淘宝/拼多多）为主，线下为辅

### 4.2 访谈大纲 - 55分钟

**【热场介绍】（5分钟）**
1. 欢迎被访者
2. 自我介绍
   - 主持人先介绍自己，然后让受访者自我介绍
   - 可以简单介绍一下您自己嘛？姓名、年龄、家庭结构（几口人）、用的是XX品牌产品

**【人群特征】（5分钟）**
- 您在购买时对产品有什么偏好？外观、功能、规格等等方面
- 在购买时除了产品本身以外，还有什么特别关注的？品牌、价格等等
- 您对这个品类的看法是什么？您对新技术的观点是什么？
- 您认为这个产品再进行哪些方面的提升是您愿意付费升级的？

**【购买情况和链路】（5分钟）**
- 您现在用的是什么品牌、什么型号的？用了多久了？
- 您记得多少钱买的吗？在哪里买的？
- 买的时候，这个产品有不同版本吗？当时怎么挑的当前这个版本？
- 您在购买时，是主要自己了解信息，还是通过他人种草推荐？你比较习惯从哪些渠道了解？
- 您在买的时候主要会关注哪些方面？哪个是您最在意的，会决定你是否购买以及花多少钱？

**【购买动机】（10分钟）**
- 除了现在用的，还有买过其他产品吗？他们的区别是什么？为什么现在用的是这个？
- 最初是什么原因了解和购买了骑行行李箱？为什么买了现在这款？是为了新功能还是什么原因？

**【使用情况】（10分钟）**
- 这个产品帮你解决了什么问题？
- 你主要在什么情况下会用？现在主要是谁在用？
- 每次出现这些使用场景的时候，你都会带去用吗？为什么有时候不会用？
- 在你说的这些场景里面，让你觉得这个箱子买得特别值得的场景是哪个？

**【使用痛点】（5分钟）**
- 您在使用过程中，遇到的主要问题是什么？骑行和智能化方面你感受怎么样？
- 这次购买体验中，你对这个产品还有哪些不满意的点？包括箱体、购买过程等等。
- 哪些点是让你比较满意的？有没有超出你预期的体验？
- 你的家人、朋友对于这个产品的评价是什么？

**【体验期望】（15分钟）**
- 目前市面上的产品，你能打几分？扣分在什么地方？
- 你当前使用这个产品，你认为他合理的价格是多少？
- 你期待他在哪些方面能做得更好？
- 如果小米做这样一个产品，你希望能解决你什么问题？
- 你最担心小米做不好的是什么？
- 你认为通过手机来帮助行李箱扩展导航、环境感知等方面的能力会奇怪吗？
- 如果有下一次购买，你预期能够花多少钱？

### 5.1 问卷模块设计

| 模块 | 问题数 | 题型 | 说明 |

| 基本信息 | 6 | 单选/多选 | 性别、年龄、收入、同住人、手机、IoT设备 |

### 5.2 问卷设计规范

- 单选题：选项互斥，包含所有可能
- 多选题：标注限选数量（如"限5项"）
- 量表题：1-10分或1-5分Likert量表
- 矩阵题：行=品牌/功能，列=评价维度
- 所有题目包含"其他，请注明"兜底选项
- 敏感信息（如收入）必须包含"不知道/不愿意透露"选项
- 未成年人（18岁以下）标记为无效问卷
