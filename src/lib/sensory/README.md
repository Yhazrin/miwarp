# Sensory UI (vendored core)

Semantic Web Audio feedback engine from [Sensory UI](https://sensory-ui.com), extracted via the shadcn registry (`sensory-ui-core`).

MiWarp uses the framework-agnostic synthesizer engine only (no React components). Playback is wired in `src/lib/services/sound-feedback-service.ts`.
